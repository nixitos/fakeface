## Что это такое

**FakeFace** — это десктопное приложение для генерации фальшивых личностей. 
Нажимаешь одну кнопку — получаешь полностью выдуманного человека со всеми данными:

- ФИО, дата рождения, адрес, школа
- Родственники (с шансом смерти)
- Нелюбимые люди, любимая и нелюбимая еда
- Хобби, травмы и болезни
- Дети, интимная информация (с возрастными ограничениями)
- Привычки, страхи, мечты
- Группа крови, знак зодиака, питомцы
- Любимая музыка и цвет

Всё в минималистичном бело-сиреневом дизайне, напоминающем сырые страницы 90-х.

---

## Как пользоваться

1. Укажи **возраст**, **пол** и **количество детей** (если возраст ≤ 17 — дети автоматически 0)
2. Нажми **[ GENERATE ]**
3. Получи готовую личность со всеми параметрами
4. Нажимай ещё — генерируются новые случайные люди

## Установка и запуск

### Готовый EXE (Windows)

1. Скачать `FakeFace.exe` из [Releases](https://github.com/nixitos/fakeface/releases)
2. Запустить — приложение готово к работе
3. *(Опционально)* Если Windows старая — установите [WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/)

### Запуск из исходников (Python)

```bash
# Клонируйте репозиторий
git clone https://github.com/nixitos/fakeface.git
cd fakeface

# Установите зависимости
pip install webview pyinstaller

# Запустите
python app.py
```

### Сборка в EXE самостоятельно

```bash
pyinstaller --noconsole --onefile --add-data "index.html;." --add-data "st.css;." --add-data "logic.js;." --add-data "faking.json;." --icon="icon.ico" --name "FakeFace" app.py
```

## Технологии

- **Python 3** — основной язык
- **pywebview** — встраивание HTML/CSS/JS в нативное окно
- **PyInstaller** — сборка в EXE
- **HTML/CSS/JS** — интерфейс и логика генерации
